# Portfoli d'Eva Is Romero

Aquest projecte és un redisseny modern del portfoli artístic de l'Eva Is Romero, utilitzant una estètica de galeria d'art contemporània ("Aurelian Gallery").

## Característiques
- **Disseny Premium**: Dark mode amb accents daurats i tipografia sofisticada.
- **Contingut Original**: Totes les imatges i textos han estat extrets de la web original.
- **Funcional**: Navegació fluida i disseny responsive per a mòbils.
- **Preparat per publicar**: Configurat per a GitHub i Netlify.

## Com publicar a Netlify (recomanat)

1. **GitHub**:
   - Crea un nou repositori a GitHub (ex: `portfolio-eva`).
   - Puja aquests fitxers (pots arrossegar-los a GitHub o fer servir la terminal).

2. **Netlify**:
   - Entra a [Netlify](https://www.netlify.com/) i inicia sessió.
   - Clica a **"Add new site"** -> **"Import from existing project"**.
   - Connecta amb el teu GitHub i selecciona el repositori `portfolio-eva`.
   - Netlify detectarà automàticament la configuració (`npm run build` i carpeta `dist`).
   - Clica a **"Deploy"** i ja està!

## Desenvolupament Local

Si vols fer canvis:
1. `npm install` (per instal·lar dependències).
2. `npm run dev` (per veure els canvis en temps real).
3. `npm run build` (per generar la carpeta `dist` definitiva).
